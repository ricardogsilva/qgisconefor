Conefor Inputs Plugin

This plugin allows the creation of input files to Conefor, a software package for studying landscape connectivity.

Further information available at:

  • Conefor homepage: www.conefor.org
  • Plugin page, with source code and bug tracker: hub.qgis.org
  • Conefor authors: Santiago Saura and Josep Torné.
